앱 접근권한 안내
                    M2 앱에서 사용하는 접근 권한에 대하여 아래와 같이 안내 해 드립니다.
                    접근권한은 필수적 접근 권한과 선택적 접근 권한으로 나누어지며, 선택적 접근 권한의 경우 허용에 동의하지 않으셔도 앱 사용이 가능합니다.

                    필수적 접근 권한
                     - 저장공간
                      기기 사진 미디어 파일 액세스 권한으로 판넬견적 다운로드, 간이투시도 제작신청(앨범에서 도면 선택 시)에 사용됩니다.

                     - 카메라
                      사진찍기 권한으로  간이투시도 제작신청(앨범에서 도면 선택 시) 사용합니다

                     안드로이드 OS 6.0 미만 버젼 스마트폰을 이용하시는 경우, 선택적 접근 권한 없이 모두 필수적 접근 권한으로 적용될 수 있습니다.
                     이 경우 운영체제를 6.0이상으로 업그레이드 하신 후 앱을 삭제 후 재설치 하셔야 접근권한 설정이 정상적으로 가능합니다.

                     기존 설치된 앱을 사용하시는 경우에는 앱을 삭제 후 재설치 하셔야 접근권한 설정이 가능합니다.

                     설정 > 애플리케이션 관리자 > M2 > 권한 메뉴에서 메뉴 설정이 가능합니다.
